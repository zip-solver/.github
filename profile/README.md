## What is Zip?

Zip is LinkedIn's daily path puzzle. You get a grid with a few numbered cells, and you draw **one continuous line** that visits every cell exactly once, passes through the numbers **in order** (1 → 2 → 3 …), and never crosses a wall.

## How to actually solve it

1. **Take the forced moves first.** A cell with only one open neighbor has to connect that way — pencil those in before anything else.
2. **Never strand a region.** If a move cuts off unvisited cells from the rest, it's wrong. Keep everything reachable.
3. **Anchor the ends.** Work outward from 1, and remember the last number has to be the final cell your line lands on.
